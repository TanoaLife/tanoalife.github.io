Tanoa Life

The home of Altis Life for Arma 3's new terrain

Welcome to the Website for Tanoa Life.

This is an Arma 3 mod that is forked from Tonic's Altis Life RPG Screenshot of Tanoa

Plan

Currently this mod is being developed on the Altis terrain so that it is feature complete by the time that Tanoa is released. When Tanoa is released the only thing left to do will be to place all of the necessary structures and rename a number of things.

Features

This is a brief but not exhaustive list of features this version of Altis Life will support

  • Vehicle Upgrades
  • Gang Housing
  • Virtual House Inventories
  • Dynamic Market
  • Black Market for Goods

Authors and Contributors

Actively being maintained by @dbenson24 Check out the repository for all the other contributors and code.

Support or Contact

As this mod is a fork from Altis Life, you should be able to get pretty good support from the Altis Life RPG Forums or by submitting an issue on the repository (if the issue is coding related)
